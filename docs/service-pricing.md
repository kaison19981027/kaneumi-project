# 調達代行サービス 料金設計

## 料金モデル：透明性のある手数料制

### 手数料率

| 調達金額 | 手数料率 |
|----------|---------|
| 〜50万円 | 8% |
| 50〜200万円 | 5% |
| 200万円〜 | 3% |

### 請求書の見せ方

| 項目 | 金額（例） |
|------|-----------|
| 資材調達価格 | 500,000円 |
| 調達手数料（5%） | 25,000円 |
| **合計** | **525,000円** |

## 方針
- 手数料は明示する（透明性で信頼を得る）
- スタートは5%で様子を見て調整
- 金額が大きいほど率を下げ、大口顧客を逃がさない

## 想定収益（月10件の場合）
- 平均調達金額100万円 × 5% × 10件 = **月50万円**
