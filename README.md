# 株式会社金海興業 — デジタル化プロジェクト

## 会社概要
- 所在地：兵庫県西播磨地域
- 従業員数：30名
- 売上規模：12億円
- Web：https://www.kaneumi.com/
- 事業：生コン・中間処理・砕石販売・土木工事・資材調達

---

## プロジェクト全体ロードマップ

### STEP 1: C — LINE × AI 自動応答（2026年 Q2）
- LINE公式アカウントにAIを連携
- 資材問い合わせ・見積を自動応答
- 技術：LINE Messaging API + Claude API

### STEP 2: A — 調達代行サービス化（2026年 Q3）
- 中小工務店・建設会社向けに調達代行を有料提供
- 手数料モデルで安定収益

### STEP 3: 社内デジタル化（並行して進める）
- Phase1：LINE WORKS導入・日報デジタル化
- Phase2：受注・配車管理（kintone）
- Phase3：請求・経理自動化（freee）

### STEP 4: B — プラットフォーム化（2027年〜）
- 建設資材の売買マッチングWebサービス
- 金海興業が西播磨の調達ハブになる
